<objective>
Ship recurring transactions per EXPN-08 + D-01-e/f/g (pending-by-default model). Two new tables: `recurring_rules` (the user-defined cadence) and `recurring_drafts` (the pending instances generated by the engine awaiting user confirmation). Engine = pg-boss cron `0 6 * * *` UTC scanning all tenants per-tenant via `withTenantTx(tenantId, system_user_id)` for active rules with `next_due_date <= today`, inserting one draft per due rule and advancing `next_due_date` via Temporal cadence math (plan 02-01's `nextOccurrence`). User actions: Confirm (insert ledger row, mark draft CONFIRMED), Edit-and-confirm (insert ledger with edited values, mark CONFIRMED), Skip (mark SKIPPED, no ledger). UI: rule CRUD form + recurring rules list + pending drafts inbox with primary-nav badge count.

Purpose: EXPN-08 is the third transaction entry-point (after manual capture in plan 02-06 and corrections in plan 02-07). Plan 02-09 search/filter consumes ledger rows minted via this flow exactly like manual ones.

Output: 2 schemas, 2 ports, 2 adapters, 7 use cases, 4 routes, 3 UI components, e2e for create-rule + confirm-draft, integration tests covering month-end Pitfall 6 + idempotent re-runs + cross-tenant RLS.
</objective>

       import { TenantId, UserId } from '@budget/shared-kernel';
       import { Temporal } from 'temporal-polyfill';
       import { nextOccurrence } from '@budget/budgeting/src/domain/cadence';
       import type { PgBoss } from 'pg-boss';

       const SYSTEM_USER_ID = '00000000-0000-0000-0000-000000000001';

       export function registerRecurringEngine(boss: PgBoss) {
         boss.work('recurring-engine', async () => {
           const today = Temporal.Now.plainDateISO();
           const tenants = await withInfraTx(async (tx) => {
             const r = await tx.execute<{ tenant_id: string }>(sql`
               SELECT DISTINCT tenant_id FROM budgeting.recurring_rules
                WHERE active = true AND next_due_date <= ${today.toString()}::date
             `);
             return r.rows;
           });
           let totalDrafts = 0;
           for (const { tenant_id } of tenants.isOk() ? tenants.value : []) {
             const r = await withTenantTx(TenantId(tenant_id), UserId(SYSTEM_USER_ID), async (tx) => {
               const rules = await tx.execute<{ id: string; next_due_date: string; cadence: string; cadence_anchor: number | null; weekly_dow: number | null; account_id: string; category_id: string | null; amount: string; currency: string; kind: string; note: string | null }>(sql`
                 SELECT * FROM budgeting.recurring_rules
                  WHERE tenant_id = ${tenant_id} AND active = true AND next_due_date <= ${today.toString()}::date
                  FOR UPDATE
               `);
               let drafts = 0;
               for (const rule of rules.rows) {
                 await tx.execute(sql`
                   INSERT INTO budgeting.recurring_drafts
                     (tenant_id, rule_id, due_date, amount, currency, account_id, category_id, kind, note, status, actor_user_id)
                   VALUES
                     (${tenant_id}, ${rule.id}, ${rule.next_due_date}, ${rule.amount}, ${rule.currency},
                      ${rule.account_id}, ${rule.category_id}, ${rule.kind}, ${rule.note}, 'PENDING', ${SYSTEM_USER_ID})
                   ON CONFLICT (rule_id, due_date) DO NOTHING
                 `);
                 const prev = Temporal.PlainDate.from(rule.next_due_date);
                 const next = nextOccurrence(
                   { cadence: rule.cadence as any, anchorDay: rule.cadence_anchor ?? undefined, weeklyDow: rule.weekly_dow ?? undefined },
                   prev,
