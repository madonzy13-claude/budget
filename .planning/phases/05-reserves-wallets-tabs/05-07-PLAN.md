The Settings UI toggle that lets the user FLIP this flag lives in Phase 6 (per D-PH5-R11). This plan only ships the cascading-hide behavior for surfaces 1 + 2, the surface-3 audit evidence, and the API serializer addition.

Lightweight plan: ~3 file edits + API serializer + 1 Vitest test + 1 audit sidecar. Parallel-eligible with Plans 05/06 since it touches different FE files (BdpTabs + column-header + layout).
