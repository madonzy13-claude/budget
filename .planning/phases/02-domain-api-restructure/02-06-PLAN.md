## TDD discipline

These tests are **already RED** — Wave 0 is implicit. The work is purely GREEN:

- For dead code (correct, history, TRANSFER): delete the code **and** the test in the same
  atomic commit. No "tombstone" — full removal. The DB has already dropped the v1.0
  columns in migration 0013; routes still pointing at them are runtime hazards.
- For aligned renames (fx_rate_date → fx_rate_as_of): rename test-side references to match
  production code.
- For env normalisation: replicate the `process.env.DATABASE_URL_APP = …replace("@db:",
"@localhost:")` pattern from `apps/api/test/routes/wallets.test.ts` line 15 at the top
  of every test file that opens its own `pg` Pool.
