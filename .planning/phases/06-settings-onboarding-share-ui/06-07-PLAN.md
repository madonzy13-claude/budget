## Trust Boundaries

| Boundary                                | Description                                                         |
| --------------------------------------- | ------------------------------------------------------------------- |
| public internet → /budgets/join/[token] | Unauthenticated, token-bearing request; the token IS the credential |
| recipient → POST accept                 | Auth-gated membership creation (Better Auth org plugin)             |

## STRIDE Threat Register

| Threat ID  | Category               | Component                                                                     | Disposition | Mitigation Plan                                                                                                                                  |
| ---------- | ---------------------- | ----------------------------------------------------------------------------- | ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| T-06-07-01 | Spoofing               | Share-link token brute force                                                  | accept      | Tokens are unguessable random (Phase 2 `createShareLink`); 7-day TTL + single-use limit the window; no Phase 6 change                            |
| T-06-07-02 | Information Disclosure | Public join page leaks budget internals                                       | mitigate    | The RSC renders only `budgetName` from `GET /budgets/join/:token` — that endpoint returns no members, no amounts; the card surfaces nothing more |
| T-06-07-03 | Elevation of Privilege | Unauthenticated visitor joins without a session                               | mitigate    | The accept POST is auth-gated server-side (`share-join.ts` 401 Unauthenticated); the unauthenticated CTA routes to sign-in first                 |
| T-06-07-04 | Tampering              | Revoked/expired token still grants access                                     | mitigate    | The accept endpoint returns 410 Revoked / 410 Expired / 409 AlreadyUsed; the card re-renders the error state and never redirects to the budget   |
| T-06-07-05 | Elevation of Privilege | Public join route accidentally inherits app auth chrome OR stays auth-bounced | mitigate    | Route placed OUTSIDE `(app)`; middleware `PUBLIC_BUDGET_PATHS` allowlist exempts `/budgets/join/*` from the PROTECTED_ROUTES bounce              |
